# Tomato_Prediction

# 🍅 Tomato Leaf Disease Prediction using Streamlit + MobileNetV2

This project predicts **tomato leaf diseases** using a **Logistic Regression** model trained on features extracted from **MobileNetV2**.

## 🚀 Features
- Uses **MobileNetV2** for deep feature extraction
- Logistic Regression classifier (91% accuracy)
- Streamlit web app for image upload & prediction
- Displays prediction confidence and class name
